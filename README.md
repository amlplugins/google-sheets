# @amlplugins/google-sheets

Native `@googleapis/sheets` SDK re-exported under the `@amlplugins` namespace.

> Google Sheets (sheets_v4).

## Install

```bash
npm install @amlplugins/google-sheets
```

## Use

```ts
import { /* named exports from @googleapis/sheets */ } from "@amlplugins/google-sheets";
// or
import sdk from "@amlplugins/google-sheets";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
